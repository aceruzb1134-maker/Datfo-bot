# DATFO Bot — Бот управления бонусными программами

Telegram-бот для автоматизации бонусных программ производителей для аптек.

## Что умеет бот

**Для администраторов DATFO:**
- Добавлять аптеки в систему (код + название)
- Загружать Excel-файл с индивидуальным планом для каждой аптеки
- Рассылать план всем зарегистрированным аптекам одной кнопкой
- Смотреть статистику ответов в реальном времени
- Отправлять напоминания тем кто не ответил
- Просматривать список согласившихся аптек

**Для аптек:**
- Регистрация по уникальному коду
- Получение индивидуального плана продаж
- Ответ одной кнопкой — Да / Нет

## Установка и запуск локально

### 1. Создайте бота

Напишите [@BotFather](https://t.me/BotFather) в Telegram:
```
/newbot
```
Скопируйте токен.

### 2. Узнайте свой Telegram ID

Напишите [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш ID.

### 3. Настройте переменные окружения

```bash
cp .env.example .env
```

Откройте `.env` и заполните:
```
BOT_TOKEN=1234567890:ABCdef...
ADMIN_IDS=123456789
```

Несколько администраторов через запятую:
```
ADMIN_IDS=123456789,987654321
```

### 4. Установите зависимости

```bash
pip install -r requirements.txt
```

### 5. Запустите

```bash
python main.py
```

## Деплой на Railway (бесплатно)

### 1. Загрузите код на GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/datfo-bot.git
git push -u origin main
```

### 2. Создайте проект на Railway

1. Зайдите на [railway.app](https://railway.app)
2. Войдите через GitHub
3. Нажмите **New Project → Deploy from GitHub repo**
4. Выберите ваш репозиторий

### 3. Добавьте переменные окружения

В Railway откройте **Variables** и добавьте:
- `BOT_TOKEN` = ваш токен
- `ADMIN_IDS` = ваш Telegram ID

### 4. Готово!

Railway автоматически запустит бота. Статус — в разделе **Deployments**.
